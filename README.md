# Iron Forge | Academia

Site institucional fictício desenvolvido como projeto de estudo com foco em estrutura profissional e boas práticas de front-end.

## ✦ Acesse

[rodrigopbarros.github.io/academia-site](https://rodrigopbarros.github.io/academia-site)

## ✦ Sobre o projeto

Landing page completa para uma academia fictícia chamada Iron Forge. Projeto desenvolvido para simular um entregável real de cliente, com estrutura de arquivos organizada e código limpo.

**Seções:**
- Hero com card de estatísticas
- Sobre com números de impacto
- Serviços oferecidos
- Planos e preços
- Formulário de contato
- Footer

## ✦ Funcionalidades

- Active link dinâmico na nav via scroll
- Fade in nas seções ao rolar a página
- Feedback visual no envio do formulário
- Layout totalmente responsivo

## ✦ Tecnologias

- HTML5 semântico
- CSS3 (Grid, Flexbox, variáveis, animações)
- JavaScript vanilla
- Fonte: Inter (Google Fonts)
